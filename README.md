# Social/Blog App

## Resources : React Native and Firebase using Cloud Firestore

## Installation
``` 
npm install
```
## Usage

- used **`expo`** 
- Install the **Expo Go** application in your phone 

## Run
Start the application using:
```
npm start
```
**OR**
```
expo start
```
And use the **Expo Go** app in your phone to run the application
